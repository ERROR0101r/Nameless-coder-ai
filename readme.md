🧐 WHAT IS NAMELESS?

Nameless is an AI chatbot with no fixed identity. Unlike other AIs that come with pre-defined personalities, Nameless is a blank slate - he becomes whatever you teach him through videos!

Built with Embedchain (simple RAG framework), Nameless learns from YouTube videos and can answer questions, write code, explain concepts, and much more based on the content you provide.

💡 The name says it all - Nameless has no identity of his own. His knowledge comes entirely from the videos YOU choose to train him with!

🚀 QUICK START GUIDE

Prerequisites

· Python 3.8 or higher
· Internet connection (for first-time setup)
· Git (optional)

One-Line Setup (Copy & Paste)

```bash
# Clone the repository
git clone https://github.com/ERROR0101r/Nameless-coder-ai.git

# Go to the folder
cd Nameless-coder-ai

# Run the smart installer (auto-installs everything!)
chmod +x setup.sh && ./setup.sh
```

That's it! The installer will:

· ✅ Check if Python is installed (install if missing)
· ✅ Check if pip is available
· ✅ Install embedchain package
· ✅ Automatically start Nameless

📦 INSTALLATION METHODS

METHOD 1: Smart Installer (Recommended)

```bash
git clone https://github.com/ERROR0101r/Nameless-coder-ai.git
cd Nameless-coder-ai
chmod +x setup.sh
./setup.sh
```

METHOD 2: Manual Installation

```bash
# Install the package
pip install embedchain

# Download the script
wget https://raw.githubusercontent.com/ERROR0101r/Nameless-coder-ai/main/app.py

# Run Nameless
python3 app.py
```

METHOD 3: Direct Python (No Git)

```python
# Just create a file called nameless.py and paste this:
from embedchain import App
nameless = App()
# Add your video URLs here...
nameless.add("youtube_video", "https://youtu.be/Y8Tko2YC5hA")
# Then start chatting
while True:
    q = input("You: ")
    print(nameless.query(q))
```
